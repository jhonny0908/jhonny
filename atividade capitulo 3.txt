8. Você está criando uma página para um blog e deseja adicionar um link para uma
outra postagem. Além disso, você quer incluir uma imagem que sirva como um
botão de "Leia mais". Qual das alternativas a seguir apresenta a opção correta para
criar o link e incluir a imagem?

a. <a href="outra-postagem.html"><img src="leia-mais.jpg" alt="Leia
mais"></a>

9. Você está desenvolvendo um formulário de registro para um site. Este formulário
precisa coletar o nome, o e-mail e a senha do usuário. Como você configuraria os
campos de entrada no HTML?

c. <form>
Nome: <input type="name" name="nome"><br>
E-mail: <input type="email" name="email"><br>
Senha: <input type="password" name="senha"><br>
</form>

10. Você está desenvolvendo o site para uma loja on-line e deseja criar uma seção
promocional com um banner que, ao ser clicado, leve os usuários para uma página
de descontos. Além disso, você quer adicionar um texto alternativo à imagem, para
promover acessibilidade. Qual seria a forma correta de implementar isso em
HTML?

d. <a href="pagina-descontos.html"><img link="banner-promocional.jpg"
alt="Descontos Incríveis"></img></a>
